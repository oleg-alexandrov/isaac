76 /*
77  This class provides the high-level logic that allows the freeflyer to
78  define the optimal inspection pose. It evaluates:
79 
80  * Visibility constraints
81  * Keepout and Keepin zones
82  * Obstacle map
83 
84  It returns a vector of possible inspection poses that can be updated
85  in case the move action fails due to planning or unmapped obstacle.
86  It also constains functions that allow inspection visualization.
87 */
